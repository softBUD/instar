## Getting Started

이 프로젝트는 인스타그램 클론 코딩 프로젝트입니다.
Next js 13 버전에 대한 학습을 목적으로 만들어졌습니다.

개발 서버 실행은 다음과 같습니다:

```bash
pnpm install

pnpm dev
```

[http://localhost:3000](http://localhost:3000) 에서 확인할 수 있습니다.
